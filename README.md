# Triangle Abstract Machine

An implementation of the Triangle Abstract Machine described by
Watt & Brown (2000). Provides an executable file called `tam-rs`.

```
Usage: tam-rs [OPTIONS] <FILE>

Arguments:
  <FILE>  Name of binary file to load

Options:
      --trace    If set, print mnemonic of instructions as they are executed
  -h, --help     Print help
  -V, --version  Print version
```

David A. Watt and Deryck F. Brown. 2000. _Programming Language Processors
in Java: Compilers and Interpreters._ Prentice Hall, London.
